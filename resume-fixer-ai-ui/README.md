# ResumeFixerAI - Frontend (React + Vite)

Quick start:

1. npm install
2. create a `.env` file with `VITE_API_BASE` pointing to your API, e.g. `VITE_API_BASE=https://resumefixeraiapi.azurewebsites.net`
3. npm run dev (for local testing) or npm run build (for production)
4. Deploy to Vercel (vercel login, then vercel)
